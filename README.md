# Operation-snackback-Build
Work-in-progress build for Operation Snackback.
Add basic installation instructions here.

To install beta builds. Download the raw zip file titled "Build_Test5" then extract the zipped folder on your device. Lastly, open up the folder and run the application called "SMP Test".

*Notice* 

The current build has smei-funtional controller support(only usable while in game not in menu). Actions that can be used by the controller can be found in the Controls file. 

There are NPCs all around the campus, they will soon be able to display information when interacted with but for now that are just there.

New functioning credits section, avaliable by clicking the credits button in the start menu. You can click the X button to go back to the start menu while in the credits.
