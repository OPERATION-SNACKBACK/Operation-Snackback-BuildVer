Work in progress build for Operation Snackback.
Add basic installation instructions here.

To install beta builds. Download the raw zip file titled "Build_Test4" then extract the zipped folder on your device. Lastly, open up the folder and run the application called "SMP Test".

*Notice* 
The current build has a semi-functional battle game(music and visuals are present). To encounter, walk into the front door of the Learning Commons. Please note that the music and visuals are the only working parts, so to leave the battle game, you need to exit the application.
